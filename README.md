## About

Feather Engine is a lightweight 2D game engine with an entity-component-system (ECS) architecture. 

Collision is currently a work in progress.

## Dependencies

* SDL2
* SDL2_image
* SDL2_ttf

## Compilation and Execution

To compile:
`make all`

To execute:
`make run`
